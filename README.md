Simple starter website

Open `index.html` in a browser to view the site locally.

Options:

- Use VS Code Live Server extension for auto-reload.
- Or run a simple static server (Python):

```bash
python -m http.server 8000
```

Files added:

- index.html — homepage
- styles.css — site styles
- script.js — tiny interaction script

Feel free to ask me to customize content, colors, or add pages.
